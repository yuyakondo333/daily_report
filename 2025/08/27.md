### Reactフックのルール
- フックは常にトップレベルで呼び出す
- 条件分岐の中でフックを使わない
- 毎回同じ順序でフックを実行する
```javascript
//  間違い
if (error) {
  return <div>Error</div>; // この後にuseEffectがあるとエラー
}
useEffect(() => {}, []);

//  正しい
useEffect(() => {}, []); // フックを先に
if (error) {
  return <div>Error</div>;
}
```
